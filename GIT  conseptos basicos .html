Git

Git init
Para iniciar el git
Git status -s
Para ver el estado del prollecto
Git add y_el_nombre_del_documento
Para guardar cambios del documento
git add .
Para guardar todos los documentos
Git commit -m "comienzo del proyecto"
Para guardar un respaldo con una importante descrupcion
git commit -am "guardar_y_espaldar"
Para guardar los cambios y hacer un respaldo
Git log --onelin
Para ver un listado de los respaldos ya hechos
Git reset --hard codigo_del_respaldo
Para volver al respaldo indicado
git commit --amend
Para abrir el editor vim
:i
Para iniciar a editar
:wq
para guardar y salir
git push -u origin main
Para subir los respaldosdos a githup
git pull
para enviar el repositorio editado en githup al editor visual
git tag nombre_de_la_version -m "descripcion"
Para guardar en el git dash la version
git push --tag
Para subir el tag a githup
git clone y_pegamos_el_url_desde_githup
Para pegar o descargar el prollecto en la maquina
git branch nombre_de_la_rama
Para iniciar una rama
git branch
Para ver las ramas
git checkout nombre_de_la_rama
para cambiar de rama
git branch -d nombre_de_la_rama
Para eliminar una rama
git merge (branch) nombre_de_la_rama
Para fucionar 2 ramas Nota: ejecutar el march desde la rama prinsipal
git push --force origin main
para forzar una actualizacion en githup
git revert HEAD o (codigo_del commit)
